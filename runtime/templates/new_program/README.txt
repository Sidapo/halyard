A Halyard program contains a number of standard directories and files:

  application.halyard: Metadata about the program.
  collects:            PLT Scheme libraries
  config:              Configuration files
  local/branding:      Icons, splash screen, etc.
  local/cursors:       Custom cursors for use in the program.
  local/graphics:      Graphics files.
  local/html:          Locally-stored HTML.
  local/media:         Locally-stored media.
  scripts:             Cards and groups in the program.
  scripts/start.ss:    The first script loaded.
  streaming/media:     Remotely-streamable media.

For more information on Halyard, see:

  http://iml.dartmouth.edu/halyard/
