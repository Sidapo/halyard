Halyard Test is an example program using the Halyard Multimedia
Runtime. For more information on Halyard, see:
<http://iml.dartmouth.edu/halyard/>

In order to run Halyard Test, you will need to have QuickTime installed. 
You can download QuickTime from:
<http://www.apple.com/quicktime/download/>

Various parts of this program require third-party software to be installed
on your computer.  For full details on what is required and how to install it, 
see:
<http://iml.dartmouthe.edu/halyard/wiki/index.php/Halyard_Test>

If you have any questions, please contact:
halyard-dev@iml.dartmouth.edu
