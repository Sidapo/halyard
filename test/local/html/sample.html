In-Engine HTML Text

This is actual HTML text, loaded off the local hard drive. It can be displayed using either Internet Explorer or the wxHtmlWindow widget (a basic-but-solid cross-platform widget normally used for online help).

Advantages

Some advantages of this widget include:

  • User stay in Tamale, improving immersion (and keeping them from getting confused about how to return to the program when they finish browsing).
  • If we use wxHtmlWindow, the browser can't fail because of IE or Netscape problems.
  • We have control over which URLs the user visits, and we can log the specific sites (and times) in the user's logfile.
